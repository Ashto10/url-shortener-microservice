SM, a url shortening microservice

Simply submit an url to sm.glitch.me/new to recieve a shortcode to that site's domain.

To use a shortcode, append it to the end of sm.glitch.me/ to be redirected.

Sites need to follow a format of http://www.example.com. Any information pass the top level domain will be discarded. Additionally, for the purposes of this assignment, websites without subdomains (ex: https://c9.io) will not work.

sm.glitch.me/new?site=https://www.google.com/ ==› returns aaaaab sm.glitch.me/aaaaab ==› redirects to https://www.google.com/

Try it out!
